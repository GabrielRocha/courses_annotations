### Course Organizer Template
A new way of the access the files of saved course on your storage.
List files and videos in the web page, using the path of the directory as the base to links.

#### Example:
##### Folder Tree
```bash
course
  ├── chapter_1
  │   └── example.mp4
  ├── chapter_2
  │   ├── example_2.mp4
  │   └── folder
  │       └── example_2_1.mp4
  └── chapter_3
      ├── example_3.mp4
      ├── image_example.png
      └── text_example.pdf
```

#### Menu Tree
``` bash
course
  ├── chapter_1
  ├── chapter_2
  │   └── folder   
  └── chapter_3
```
